# Real-Time Stock Analyzer 

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/forbiddenscholar/real_time_stock_analyzer.git
cd real_time_stock_analyzer
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run Dashboard

**Linux:**
```bash
cd src
streamlit run ../dashboard/app.py
```

**Windows:**
```bash
cd src
streamlit run ../dashboard/app.py
```
